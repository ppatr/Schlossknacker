# Schlossknacker
An diesem Projekt haben Arne, Calvin und Patrick aus der ITA19a teilgenommen.

Projektbeschreibung:
Durch drehen an einem Potentiometer muss ein Schloss geknackt werden (ähnlich Tresor), Hinweise werden durch eine LED gegeben.
